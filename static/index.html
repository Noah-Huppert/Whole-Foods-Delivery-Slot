Shopper

Overview

The shopper tool provides the following functionality:

  1. Cart management
  2. Delivery slot availability notification
The 2 sections below labeled Cart and Delivery are user interfaces which provide the above functionality.

See the corresponding sections for a more detailed description of their functionality.

Items

Provides an interface to manage items in your cart.

If an item is not in your cart a search of the Whole Foods website can be performed and the item can be added to your cart.

Items which are removed from your cart will be archived, but can still be viewed in the "Archived Items" part of this interface. These items can be easily added back to your cart at any time.
